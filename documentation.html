    Inscription utilisateur

      Outils

      • Chemin d'acces : /user/singnin/
      • Methode POST

        • Informations & name

        • Nom : de type alphabet et pour name 'name'
        • Prenom : de type alphabet et pour name 'lastName'
        • email : de type email et pour name 'email'
        • age : de type numerique et pour name 'age'
        • adresse : de type alpha-numerique et pour name 'adresse'
        • pays : de type alphabet et pour name 'pays'
        • sexe : de type alphabet et pour name 'sexe'
